# TupperStash  
A simple and fun way to organize the things you don’t want to lose — notes, ideas, memes, or anything else deserves its own little **tupper**

When you want to separate your gallery into sections, save memes for later, or keep small things together in a clean way, TupperStash gives you a quick and lightweight space to stash them.


**Main Features :**  Create custom tupper collections, add text items, add meme/image items from the internet, delete or rename collections, clean two-activity navigation, Room database CRUD operations, multiple RecyclerView item layouts, interface-based click handling, custom dialogs, custom views, Retrofit API integration, LiveData + Coroutines, smooth and simple UI.
